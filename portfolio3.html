To do list

The primary functionality of this web application is to allow users to create and manage tasks. Users can input task descriptions and dynamically add tasks to the list by pressing the "Enter" key.

Each task is displayed as a list item, and users can mark them as completed by clicking on them. Additionally, tasks can be individually removed from the list using their respective delete buttons.

The application also features real-time updates. A dynamic clock accurately displays the current time and updates every second to maintain precise timekeeping. Furthermore, the application utilizes the Weatherstack API to fetch real-time weather data and display the current temperature.
